# **App Name**: ArithmaGen

## Core Features:

- Mathematical Functionalities: Implement basic arithmetic (addition, subtraction, multiplication, division) and advanced mathematical calculations (derivatives, integrals, vector math, coordinate transformations).
- Point Input/Output: Allow users to input points manually and export calculated results or functions along with their solutions. Format options: .CSV or similar.
- Line Creation and Analysis: Enable users to define lines, calculate bearings and distances, and compute misclosures for loops. Provides a 'tool' using LLM reasoning when to include tolerances based on provided or discovered standards.
- Graphical Loop Plotting: Visualize basic loops based on user-defined line data.
- Annotation and Labeling: Enable users to add labels to lines, text elements, and other graphical components.
- Homepage Navigation: Create an intuitive homepage with clearly defined links to all functionalities.

## Style Guidelines:

- Light Mode: Background color will be light gray (#F0F0F0), almost white.
- Dark Mode: Primary color is deep green (#1A4D2E) to provide a comfortable dark experience.
- Accent: Use yellow-green (#82ff6f) for interactive elements in dark mode, providing contrast and highlighting CTAs.
- Body and headline font: 'Inter', sans-serif, for clear readability and a modern look.
- Use minimalist, geometric icons for mathematical functions and actions. Use a consistent stroke weight and style across all icons.
- Implement a clean, grid-based layout to maintain visual consistency and ease of navigation. Use whitespace effectively to separate sections.
- Incorporate subtle animations on interactive elements such as button hovers and transitions to enhance user engagement without being distracting.
